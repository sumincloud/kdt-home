1. @for문

@for 키워드는 스타일을 반복적으로 출력하는데 사용됩니다.

javascript의 for문과 유사합니다.

    문법)
    trough:종료 만큼 반복(종료값도 포함)

    @for $변수 from 시작 through 종료 {
      //반복할 내용
    }

    문법2)
    //to : 종료 전까지 반복(종료값 미포함)

    @for $변수 from 시작 to 종료{
      //반복할 내용
    }

$변수 => #{변수}

#{변수}는 코드 어디든지 변수값을 사용할 수 있다.

    적용 예)
    $img: 'example.png';
    $imgPath: unquote('./img/#{$img}');
    unquote는 따옴표를 제거하고 저장해줌

    .selector {
      background-image: url(#{$imgPath});
    }
