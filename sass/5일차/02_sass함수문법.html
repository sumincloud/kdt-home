02_sass함수문법

@function 문법

  • 사용자가 함수를 직접 정의하여 사용할 수 있습니다.
  • @mixin과 비슷해 보이지만 반환되는 내용이 다릅니다.
  • @mixin은 지정한 스타일을 반환하지만 @function(함수)는 연산된 값을 @return키워드로 반환합니다.
    문법)
    @mixin 믹스인명($매개변수){
      스타일 서식
    }
    ----------------------------
    @function 함수명($매개변수){
      @return 값;
    }
    ----------------------------
    @minin기법은 호출시 '@include 믹스인이름(인수);' 처럼 사용
    함수문법은 호출시 '함수명(인수)'으로 호출하여 사용한다.
